# TaskHub

TaskHub é uma aplicação simples de gerenciamento de tarefas, construída com Next.js.

Os usuários podem adicionar novas tarefas, marcá-las como importantes, excluí-las ou movê-las para a lixeira e claro marcá-las como concluidas. Além disso, os usuários podem filtrar suas tarefas por status: *Todas*, *Importantes* ou *Lixeira*.

## Como Visualizar

Você pode visualizar este projeto online clicando [aqui](https://task-hub-cyan.vercel.app/) ou clonando este repositório e executando o projeto em seu navegador. Para executar o projeto localmente, siga estas etapas:

1. Clone este repositório usando o comando:

```
git clone https://github.com/arisonfirmino/task-hub.git
```

2. Navegue até o diretório do projeto.
3. Instale as dependências com o comando:

```
npm install
```

4. Inicie o servidor local com o comando:

```
npm run dev
```

5. Abra seu navegador e acesse [http://localhost:3000](http://localhost:3000).

## Componentes

### Nav

O componente de navegação permite que os usuários filtrem suas tarefas por status.

### AddNewTask

Este componente permite aos usuários adicionar uma nova tarefa à lista de tarefas.

### Task

O componente de tarefa exibe uma única tarefa na lista. Os usuários podem marcar a tarefa como concluída, marcar como importante, excluir a tarefa ou restaurá-la da lixeira.

### Search

O componente de pesquisa permite aos usuários buscar tarefas na lista.
